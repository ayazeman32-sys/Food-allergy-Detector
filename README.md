# 🍔 Food Allergy Detector

A simple desktop application built with **Python** and **PyQt5** that helps users determine whether a selected food is safe to eat based on their allergies.

## 📌 Project Description

Food Allergy Detector allows users to:

- Select a food item from a list.
- Choose one or more food allergies.
- Check whether the selected food contains ingredients related to those allergies.
- Display a clear result indicating whether the food is safe or not.

This project was developed as an **AI/ML Internship Project** to demonstrate Python programming, GUI development, and basic allergy detection logic.

---

## ✨ Features

- User-friendly graphical interface (GUI)
- Food selection using a dropdown menu
- Multiple allergy selection using checkboxes
- Instant allergy detection
- Displays:
  - ✅ Safe to Eat
  - ❌ Not Safe (Shows detected allergens)

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/ayazeman32-sys/Food-allergy-Detector.git
```

### 2. Open the project folder

```bash
cd Food-allergy-Detector
```

### 3. Install PyQt5

```bash
pip install PyQt5
```

### 4. Run the application

```bash
python main.py
```

---

## 🖥️ How It Works

1. Select a food item.
2. Select your allergies.
3. Click **Check Food**.
4. The application compares the selected food's ingredients with the chosen allergies.
5. The result is displayed instantly.
